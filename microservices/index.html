The Microservices

msa-httpbin

HTTP Request & Response Service. Useful for testing HTTP REST tools.

msa-faker

Fake data generator. Can generate names, addresses, phone numbers, etc.

msa-corpora

A collection of small corpuses of interesting data.

msa-world-factbook

Semi structured API for the CIA World Factbook, a reference of information for 267 countries.

msa-fakeimage

Dynamic image generator. Can generate png, jpg and gif images at various resolutions.

msa-string-similarity

Various algorithms to mesure the similarity of N strings.

msa-geohash

Base32 Geoashing as a service, with geojson support.

msa-geocoder

Reverse geocoding as a simple offline service.
